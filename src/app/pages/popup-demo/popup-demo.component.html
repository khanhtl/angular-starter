PopupService API

Access the global popup registry via the simplified async API.

open(key, config)

Opens a popup and returns a ref. Useful for manual control or persistent dialogs.

openAndWait(key, config)

The preferred way. Opens a popup and returns a Promise that resolves when the user closes it.

unregister(key)

Unregisters a popup from the registry. Always use in ngOnDestroy.
