app-tag-box API Reference

Property Type Default Description
label string '' The label text for the component.
hint string '' Helper text displayed below the component.
errorText string '' Error message displayed when the component is invalid and touched.
required boolean false Whether the field is required.
disabled boolean false Whether the component is disabled.
readonly boolean false Whether the component is in read-only mode.
dataSourceInput T[] | Store | DataSourceOptions [] Data source configuration. Supports arrays, CustomStore, or local data.
displayExpr string '' The property used to display item text.
valueExpr string 'id' The property used as the actual value.
dropDownWidth string | number 'auto' Width of the dropdown list. If 'auto', it matches the input width.
searchEnabled boolean false Whether to show the search input in the dropdown.
clearable boolean false Shows a clear all button when items are selected.
showSelectAll boolean false Shows a "Select All / Deselect All" option in the dropdown.
maxDisplayedTags number 3 Maximum number of tags to display before showing "+N more".
closeOnScroll boolean false Whether to close the dropdown when the page is scrolled.
groupExpr string '' Property name used to group items in the dropdown.
groupsCollapsed boolean false Whether groups are collapsed by default.

Outputs

Property Type Description
selectionChanged EventEmitter<SelectedItemInfo[]> Emitted when selection changes. Provides detailed info about all selected items.

Methods

Method Returns Description
getSelectedItemsInfo() SelectedItemInfo[] Returns an array of objects containing the raw item, its value, and its display text.
getSelectedValues() any[] Returns an array of the current selected values.
getSelectedDisplayTexts() string[] Returns an array of the display texts of all selected items.
getSelectedItems() any[] Returns the raw objects of all selected items.
selectByValues(values: any[]) void Programmatically sets the selection by providing an array of values.
clearSelection() void Clears all selected items.

Templates

Directive Description
itemTemplate Customize each row in the dropdown list. Context: item, index.
tagTemplate Customize the display of selected items as tags. Context: item.

Usage Example

<app-tag-box
  [dataSourceInput]="items"
  displayExpr="name"
  valueExpr="id"
  [searchEnabled]="true"
  [showSelectAll]="true"
  [clearable]="true"
  [maxDisplayedTags]="5"
  [(ngModel)]="selectedValues">

  <ng-template itemTemplate let-item>
    <div class="custom-item">
      {{ item.name }}
    </div>
  </ng-template>

  <ng-template tagTemplate let-item>
    <span>{{ item.name }}</span>
  </ng-template>

</app-tag-box>

Key Differences from Select Box

Feature Select Box Tag Box
Selection Mode Single Multiple
Value Type Single value Array of values
Display Text / fieldTemplate Tags with remove buttons
Select All Not available Optional showSelectAll
Tag Limit N/A maxDisplayedTags with overflow count
